## üì¶ Description

**YAIM904 d16b ‚Äì Electricity without Pollution** removes **pollution generation from all entities that use electricity**.

This MOD is designed for players who want to decouple electrical power usage from environmental impact, allowing large electric-based factories to operate without increasing pollution levels.

It is especially useful for clean-energy playstyles, testing environments, or factories focused on logistics and automation rather than combat pressure.

---

## ‚öôÔ∏è How it works (Player view)

* All electrically powered entities **stop generating pollution**
* Energy consumption remains unchanged
* No configuration or player interaction is required
* The change is applied automatically once the MOD is enabled

Only pollution behavior is modified; normal gameplay flow remains intact.
